## Welcome to UCC Control de Acceso

UCC Control de Acceso is a comprehensive access management system designed for Universidad Cooperativa de Colombia (UCC) to manage contingent entry for users who do not have their physical ID card.

<Note>
This system is **ONLY** used when users do not carry their physical ID card. It does not register normal access entries.
</Note>

## How It Works

The system manages three types of failure incidents:

1. **Forgot Card (Olvido)** - Allows 3 entries, blocks on the 4th occurrence
2. **Lost Card (Pérdida)** - Immediate blocking, user must visit administration
3. **Stolen Card (Robo)** - Immediate blocking, user must visit administration

## User Roles

The system supports three primary user roles:

- **Students (Estudiantes)** - Academic program members with faculty and semester information
- **Employees (Empleados)** - University staff with department and position details
- **Contractors (Contratistas)** - External service providers with company and contract information

<Info>
Users can have multiple roles simultaneously. For example, a student can also be an employee.
</Info>
